# smhvz

GitHub Pages için hazırlanmış kişisel portföy sitesi.

## Düzenlenecek Alanlar

- `index.html`: İsim, açıklama, proje kartları ve iletişim linkleri.
- `styles.css`: Renkler, aralıklar ve responsive tasarım.
- `assets/portfolio-visual.png`: İlk ekranda kullanılan görsel.

GitHub Pages kaynağını repository ayarlarında `main` branch ve root klasör olarak seçebilirsin.
