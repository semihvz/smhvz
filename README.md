# smhvz blog

GitHub Pages üzerinde çalışan statik blog sitesi.

## Yeni Makale Ekleme

1. `posts/` klasöründeki mevcut bir yazıyı kopyala.
2. Dosya adını kısa ve URL uyumlu yap: `ornek-yazi.html`.
3. Başlık, tarih, kategori, okuma süresi ve içerik alanlarını düzenle.
4. `index.html` içindeki `article-grid` bölümüne yeni makale kartı ekle.
5. `feed.xml` dosyasına yeni yazı için bir `item` ekle.
6. Değişiklikleri commit edip push et.

## Dosya Yapısı

- `index.html`: Blog ana sayfası, arama, filtreler ve makale listesi.
- `posts/`: Yayınlanan makale sayfaları.
- `styles.css`: Tüm site ve makale sayfası stilleri.
- `script.js`: Tema değiştirme, arama ve kategori filtreleri.
- `feed.xml`: Basit RSS yayın akışı.

GitHub Pages kaynağı `main` branch ve root klasör olmalı.
